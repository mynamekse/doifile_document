# 2.2 Farming System (Doi Farming)

*   **Crops:**
    *   *Main:* ชา (Assam), กาแฟ (Arabica) - ไม้ยืนต้น เก็บเกี่ยวได้หลายครั้ง
    *   *Seasonal:* ผักกาด, สตอเบอร์รี่, ดอกเมืองหนาว
    *   *Herbs:* มะกรูด, ขิง, ตะไคร้ (ใช้แปรรูป/ทำอาหาร)
*   **Mechanics:**
    *   **Terracing:** การปลูกพืชบนพื้นที่ลาดชัน มีผลต่อการจัดการน้ำ
    *   **Soil Quality:** ต้องบำรุงดินดอยให้เหมาะกับพืชเศรษฐกิจ
